📦 Materials Required

List of Materials

  • 🛢️ Plastic Drum / Container
  • 🔩 Compression Lever
  • 🧱 Brick Mold
  • 🏷️ Label Stickers
  • ♻️ Plastic Waste (Raw Material)

Description

These materials are used to build the Eco-Brick system. Plastic waste is collected and compressed using a lever inside a mold to form eco-friendly bricks.
